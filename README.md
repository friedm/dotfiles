# Dotfiles

Configuration files for development environment

### Setup

* Clone to (hidden) directory under home
* `cd <clone dir>`
* `./setup_git_prompt.sh`
* `./link_rc_files.sh`
* `./setup_vim.sh`

### Versioning new rc files

* Copy the rc file into `<repo root>/dotfiles/`
* re-run `./link_rc_files.sh`
